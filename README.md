# Nobano Anfrageformular

Statische Kundenformular-Seite für die Nobano WhatsApp-Auftragserfassung.

- Die Seite wird über GitHub Pages bereitgestellt.
- Suchmaschinen werden mit `noindex` und `nofollow` ausgeschlossen.
- Vorausgefüllte Kundendaten stehen ausschließlich im URL-Fragment nach `#` und werden nicht an GitHub übertragen.
- Jeder Link erhält eine zufällige Vorgangsnummer.

Hinweis: Die Seite ist nicht gelistet, aber technisch für jede Person erreichbar, die den vollständigen Link besitzt.
